# SCREEN 6 — ONBOARDING FLOW (NEW)

## Route: Shown on first app launch only

## 6.1 PURPOSE

Welcome new users, explain the app philosophy, collect initial preferences, and guide them to their first recording.

## 6.2 FLOW — 5 SCREENS

```
[1. Welcome] → [2. How It Works] → [3. Choose Language] → [4. Choose Starting Point] → [5. Ready!]
```
